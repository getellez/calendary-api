# calendary-api
An app to handle your schedule

# Requirements
- Node.js v16 LTS

# Run
```
cd calendary-api
npm install 
npm run dev
```

# Environments
You can assign 1 of 3 options to the NODE_ENV variable in the `.env` file.

- NODE_ENV=development
  - This option is used by the developers in theirs computers
- NODE_ENV=demo
- NODE_ENV=production
